react-redux-todo
===============
预览地址：[redux-todo](https://funlee.github.io/react-redux-todo/ "react-redux-todo")

安装
----
```bash
git clone https://github.com/funlee/react-redux-todo.git
cd react-redux-todo
npm install
npm run start
```
然后在浏览器里输入：http:localhost:8080 即可访问

主要功能
-------
* Task 增加、删除
* 增加 Task 支持回车（Enter）
